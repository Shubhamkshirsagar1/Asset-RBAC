# Phase 5: Project-Management Domain — Implementation Plan

> REQUIRED SUB-SKILL: superpowers:executing-plans. Steps use checkbox syntax.

**Goal:** Add a second domain — **Projects** and **Tasks** — on the *same* engine, guards, and
scope machinery. The point is to prove the system is domain-agnostic: different resources, different
roles, identical authorization code.

**Architecture:** Identical pattern to Phase 4 — list endpoints use `requirePermission` + the
`scope-where` helper; single-record endpoints use `requireOwnership`. Project owner field =
`ownerId`, org field = `orgUnitId`. Task owner field = `assigneeId`, no org field. Demonstrates a
status-based condition (`task:update` only while not `done`).

**Tech Stack:** existing.

---

## Endpoints

**Projects** (`/projects`, resourceType `project`):
| Method | Path | Guard |
|--------|------|-------|
| GET | `/projects` | `requirePermission('project','read')` |
| POST | `/projects` | `requirePermission('project','create')` (ownerId = caller) |
| GET | `/projects/:id` | `requireOwnership('project','read', load)` |
| PATCH | `/projects/:id` | `requireOwnership('project','update', load)` |

**Tasks** (`/tasks`, resourceType `task`):
| Method | Path | Guard |
|--------|------|-------|
| GET | `/tasks` | `requirePermission('task','read')` |
| POST | `/tasks` | `requirePermission('task','create')` |
| GET | `/tasks/:id` | `requireOwnership('task','read', load)` |
| PATCH | `/tasks/:id` | `requireOwnership('task','update', load)` |
| POST | `/tasks/:id/complete` | `requireOwnership('task','complete', load)` |

## Task 1: services

```javascript
// project.service.js
import { models } from '../db/index.js';
const { Project } = models;
const err = (m, s) => Object.assign(new Error(m), { status: s });

export const listProjects = (where) => Project.findAll({ where, order: [['name', 'ASC']] });
export const findProject = (id) => Project.findByPk(id);
export async function createProject(data, userId) {
  if (!data.name) throw err('name is required', 400);
  return Project.create({ name: data.name, orgUnitId: data.orgUnitId ?? null, ownerId: userId });
}
export async function updateProject(project, data) {
  for (const f of ['name', 'orgUnitId']) if (data[f] !== undefined) project[f] = data[f];
  await project.save();
  return project;
}
```

```javascript
// task.service.js
import { models } from '../db/index.js';
const { Task } = models;
const err = (m, s) => Object.assign(new Error(m), { status: s });

export const listTasks = (where) => Task.findAll({ where, order: [['title', 'ASC']] });
export const findTask = (id) => Task.findByPk(id);
export async function createTask(data) {
  if (!data.projectId || !data.title) throw err('projectId and title are required', 400);
  return Task.create({ projectId: data.projectId, title: data.title, assigneeId: data.assigneeId ?? null, status: data.status ?? 'todo' });
}
export async function updateTask(task, data) {
  for (const f of ['title', 'assigneeId', 'status']) if (data[f] !== undefined) task[f] = data[f];
  await task.save();
  return task;
}
export async function completeTask(task) { task.status = 'done'; await task.save(); return task; }
```

## Task 4: pm e2e (`test/pm.e2e.test.js`)

Setup: a **pm** tenant; workspace org unit; roles:
- `member` grants: `project:read:own`, `project:create:any`, `task:read:own`, `task:create:any`,
  `task:update:own` **condition** `{ "resource.status": { "ne": "done" } }`, `task:complete:own`.
- `lead` grants: `project:read:any`, `task:read:any`.
Users: `dev` (member), `lead` (lead). Projects: `pOwn` (owner dev), `pOther` (owner lead).
Tasks: `tOwn` (assignee dev, status todo), `tDone` (assignee dev, status done).

Scenarios:
1. `dev` `GET /projects` → only `pOwn` (own scope).
2. `lead` `GET /projects` → both (any scope) — same engine, different domain.
3. `dev` `PATCH /tasks/:tOwn` `{title}` → 200 (own + not done).
4. `dev` `PATCH /tasks/:tDone` → 403 (status-condition: cannot edit a done task).
5. `dev` `POST /tasks/:tOwn/complete` → 200 (status → done).
6. `dev` `GET /projects/:pOther` → 403 (not own).
